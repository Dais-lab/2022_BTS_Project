머신러닝 모델 학습

머신러닝 모델 학습 방법

1. 우리가 가진 데이터를 training data와 test data로 나누어 줌

2. 초기 모델에 training data를 넣고 학습함

3. test data로 잘 맞는지 검증함

4. 모델의 일반화 성능을 측정함

5. 결과를 개선하기 위해 모델을 수정함

딥러닝 모델 학습

딥러닝 모델 학습 방법

1. 학습용 Feature 데이터를 입력하여 예측값 구함(순전파)

2. 예측값과 실제값 사이의 오차 구함( Loss 구함)

3. Loss를 줄일 수 있는 가중치 업데이트 함(역전파)

4. 1~3번의 반복으로 Loss를 최소로 하는 가중치 얻음
